
# Deep Recurrent Survival Ranking (DRSR)

A `tensorflow` implementation of DRSA model. This is the experiment code for our SIGIR 2020 paper "[A Deep Survival Model for Unbiased Ranking](https://jinjiarui.github.io/)".
